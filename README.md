# 차트 관리 시스템

차트를 직접 커스텀할 수 있는 시스템 구축하기.

## 기술 스택

- React, Redux, React Query, eChart, sass, storybook, react-testing-library(jest-dom)

## React Test

- Test를 위해 react-testing-library를 사용하였다.
- test를 위해서 yarn test or npm run test를 입력한다.
